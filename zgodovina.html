Prvi pomnilnik: Prva praktična oblika
delovnega pomnilnika je nastala
leta 1947, imenovana je bila:
Williams Tube
Pomnilnik je bil zgrajen iz
katodne cevi, na kateri so se
shranjevali podatki.
Kapaciteta pomnilnika je bila
približno med 1024-2560 bitov.
Frederic C. Williams: Rojen 26. junija, 1911.
Bil je angleški inženir,
ki je ustvaril prvi
bralno pisalni pomnilnik,
ki je zagnal prvi program
na računalniku
Manchester baby.
Delovanje: Podatki so se shranjevali kot električno nabita mesta.
Ta pojav se imenuje: "sekundarna emisija"
Postopek ustvarjanja nabojev se uporablja kot zapisovanje bitov.
Ta nabita mesta so se prikazovala kot pike, ki so se brisale in pisale.
Ena vrstica pik predstavlja besedo.
Ker se je dalo elektronski žarek premikati kamor koli po cevi,
lahko računalnik dostopa do katerega koli mesta.
To imenujemo "Random access memory", ki ga še danes uporabljamo.
Prvi SDRAM: Sinhronično dinamični pomnilnik
SDRAM oz. SDR RAM,
ki spada pod
DRAM-e, je bil narejen leta 1970.
Naredili so ga podjetje Intel.
Bil je integrirano vezje,
zato je bil sinhroniziran s
samim procesorjem.
Delovanje SDRAM-a: Pomnilniki SDRAM so
bili na začetku,
SDR pomnilniki:
"Single data rate"
kar pomeni, da so se
podatki prenašali samo z
enim taktom.
Njihove kapacitete so
kolebale med 2kb-16Mb.
